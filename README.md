# Simple Classification Cat Dog Model using DL CNN

### Cat vs Dog Classification using CNN
#### Project Description
This project is a simple image classification system that identifies whether an image contains a cat or a dog using a Convolutional Neural Network (CNN).
It is a beginner-friendly deep learning project created to understand CNN basics and image classification.

#### Features

Classifies images as Cat or Dog
Uses CNN for image learning
Image preprocessing and normalization
Easy to train and test

#### Technologies Used

Python
TensorFlow / Keras
NumPy
OpenCV / PIL
